Tank Blast Arena 2D

2017 - 2019


Play on simmer.io

Genre: Tank shooter, 2D top-down view.
Description: Multiplayer game supporting up to 20 concurrent users (CCU). Deathmatch mode, aka free-for-all. It includes AI-driven tanks (in gray color) to compensate for the absence of players. The map was built using layered tiles. Another features are:

  • Basic personalization (player name and tank color).
  • Pickups (health, shield, laser and speed boost).
  • Mini-map.
  • Leaderboard.

Developed with: Unity/C# and Photon PUN 2 (multiplayer plugin).
Target platform: Web browser on PC.
